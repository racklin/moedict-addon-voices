Moedict-Desktop 語音擴充套件
-----------------------------
這是 Moedict-Desktop 的語音擴充套件。
包含 國語 ／ 閩南 ／ 客家 三個語言。

Build
-----------------------------

1. 下載語音檔案 `fetch_moe_voices.sh`
2. run build.sh
